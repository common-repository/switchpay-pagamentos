## Ofereça Cartão, Pix e Boleto a seus clientes

Com o plugin da SwitchPay, você oferece mais facilidade para o seu cliente e conta com as melhores taxas do mercado.

Disponibilize os principais meios de pagamento para o consumidor, como o **cartão de crédito** à vista ou parcelado em até 12x, **boleto bancário** e o **Pix**!

#### **Algumas vantagens**

- Configure valor mínimo para parcelar.
- Configure a quantidade de parcelas sem juros.
- Configure a cobrança de juros.
- Pix com tempo de pagamento de 30 minutos, liberando seu estoque rapidamente.
- Pix com operações disponíveis 24 horas por dia todos os dias da semana

== Installation ==

= Plugins necessários =

- WooCommerce 3.0 ou posterior para funcionar;
- Brazilian Market on WooCommerce para funcionar;

1 - Instale usando o instalador de plugins do WordPress, ou envie os arquivos do plugin para a pasta wp-content/plugins no seu servidor.
2 - Ative o plugin.
3 - Instale o plugin Brazilian Market on Woocommerce.
4 - Desativar a opção Manter Estoque (minutos) do WooCommerce.

= Como fazer para ativar =

#### **SwitchPay**

- [Crie sua Conta SwitchPay](https://switchpay.app.br/cadastro/vendedor), é grátis e fácil!


#### **WooCommerce**

- Configure o plugin na aba WooCommerce -> Configurações -> pagamentos -> SwitchPay;
- Selecione o ambiente, na SwitchPay temos o ambiente para testes e o ambiente de Produção, onde pode ser alterado escolhendo entre eles;
- Insira suas credenciais (tokens ID e Client Secret);
- Para ambiente de testes disponibilizamos as credenciais, basta solicitar no email contato@switchpay.com.br;

= Qual é a licença do plugin? =

= O que eu preciso para utilizar este plugin? =

* Possuir uma conta na [SwitchPay] (https://switchpay.com.br).
* WooCommerce 3.0 ou posterior.
* Brazilian Market on Woocommerce.
* Desativar a opção Manter Estoque (minutos) do WooCommerce.

= Consigo habilitar somente o cartão de crédito, boleto ou PIX? =

Sim, você poderá habilitar os três meios de pagamento ou apenas um deles, na configuração padrão do WooCommerce em WooCommerce -> Configurações -> Pagamentos -> SwitchPay.
